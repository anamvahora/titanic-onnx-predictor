# Titanic Survival Predictor

This project is a deep learning model built using PyTorch to predict whether a passenger survived the Titanic disaster. 
The model was trained on the Titanic dataset, converted to ONNX format, and deployed in the browser using HTML and JavaScript with ONNX Runtime.
Users can enter passenger details like age, class, and fare, and the model will give a prediction along with a confidence score in real time.
